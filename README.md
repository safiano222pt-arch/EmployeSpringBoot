# Employee API - Spring Boot REST CRUD

Une API REST simple et complète pour gérer des employés, développée avec **Spring Boot 3**, **Spring Data JPA**, **MySQL** et **Lombok**.

## Fonctionnalités

- Créer un employé (POST)
- Lister tous les employés (GET)
- Récupérer un employé par ID (GET)
- Mettre à jour un employé (PUT)
- Supprimer un employé (DELETE)
